# <a name="compatibility-assessment-in-desktop-analytics"></a>데스크톱 분석의 호환성 평가

> [!Note]  
> 이 정보는 정식으로 발표 되기 전에 대폭 수정 될 수 있는 미리 보기 서비스에 연결 합니다. Microsoft는 여기에 제공된 정보와 관련하여 명시적이거나 묵시적인 어떤 보증도 하지 않습니다.  

Windows Analytics에서 업그레이드 평가 된 일반, 예를 들어: 주의가 필요한 또는 수정을 사용할 수 있습니다. 모든 시각적 표시기 앱 또는 문제가 있는 드라이버 우선 순위를 지정 하거나 insights를 업그레이드 하는 방법은 제공 하지 않습니다. 사용 하 여이 기능을 대체 하는 데스크톱 분석 **호환성 위험**합니다. 데스크톱 Analytics 업그레이드 전 시나리오에 대 한 배포 보기에만 앱에 대 한 평가 보여 줍니다. 현재 배포 계획에 포함 된 컴퓨터에서 Microsoft 가져옵니다 정보에 따라 앱을 분류 합니다.

데스크톱 분석에는 다음 호환성 평가 범주를 사용 합니다.

- **낮음**: 서비스를 찾을 수 없는 신호는 Windows에 대 한이 응용 프로그램 위험에 처를 업그레이드 합니다. 대상 OS로 작업할 가능성이-됩니다.  

- **중간**: 수정 가능성이 높은 수는 있지만 분석 응용 프로그램 수 기능을 장애가 있는 나타냅니다.  

- **높은**: 응용 프로그램은 거의 확실히 중 또는 업그레이드 후에 실패 합니다. 이 수정을 해야 합니다.  

- **알 수 없음**: 앱 평가 되지 않았습니다. 와 같은 다른 정보는 *MS 알려진 문제* 하거나 *준비에 대 한 Windows*합니다.  

배포 계획의 앱 또는 드라이버 자산 목록에서 각 자산에 대 한이 값이 표시 됩니다는 **호환성 위험** 열입니다.


## <a name="app-risk-assessment"></a>앱 위험 평가

데스크톱 분석 응용 프로그램에 대 한 평가 등급을 생성 하는 데 사용 하는 여러 원본 가지가 있습니다.

- [Microsoft의 알려진 문제](#microsoft-known-issues)
- [Windows 카탈로그에 대 한 준비](#ready-for-windows)
- [고급 정보](#advanced-insights)

데스크톱 Analytics에서 앱의 각 원본에 대 한 평가 찾을 수 있습니다. 배포 계획의 앱 자산 목록에서 해당 속성 플라이 아웃 창을 열려면 개별 앱을 선택 합니다. 전체 권장 사항 및 평가 수준을 볼 수 있습니다. 합니다 **호환성 위험 요소** 섹션에서는 이러한 평가 대 한 세부 정보를 보여 줍니다.


## <a name="microsoft-known-issues"></a>Microsoft의 알려진 문제

데스크톱 분석 알려진된 문제에 대 한 Microsoft 응용 프로그램 호환성 데이터베이스에 살펴봅니다. 이 데이터베이스를 사용 하 여 Microsoft에서 공개적으로 사용 가능한 응용 프로그램에 대 한 모든 기존 호환성 블록 또는 다른 게시자를 확인 합니다. 이 검사는 선택 하면 배포 계획에 대 한 대상 OS에만 적용 됩니다.

앱 속성 창에 다음과 같은 문제를 볼 **알려진 문제는 MS**:

### <a name="application-is-removed-during-upgrade"></a>응용 프로그램을 업그레이드 하는 동안 제거

Windows는 호환성 문제를 검색 합니다. 응용 프로그램은 새 OS 버전으로 마이그레이션되지 않습니다. 조치가 계속 하려면 업그레이드에 필요 하지 않습니다.

### <a name="blocking-upgrade"></a>업그레이드를 차단

Windows는 차단 문제를 감지 하 고 업그레이드 하는 동안 응용 프로그램을 제거할 수 없습니다. 새 OS 버전에는 작동 하지 않을 수 있습니다. 업그레이드 하기 전에 응용 프로그램을 제거 합니다. 다시 설치 하 고 새 OS 버전에서 테스트 합니다.

### <a name="blocking-upgrade-but-can-be-reinstalled-after-upgrading"></a>업그레이드를 차단 하지만 업그레이드 한 후 다시 설치 수

응용 프로그램 새 OS 버전과 호환 되지만 마이그레이션되지 않습니다. Windows를 업그레이드 하기 전에 응용 프로그램을 제거 합니다. 새 OS 버전에 다시 설치 합니다.

### <a name="blocking-upgrade-update-application-to-newest-version"></a>최신 버전으로 응용 프로그램 업데이트 업그레이드 차단

기존 버전의 응용 프로그램의 새로운 OS 버전과 호환 되지 않습니다 및 마이그레이션되지 않습니다. 응용 프로그램의 호환 되는 버전은 사용할 수 있습니다. 업그레이드 하기 전에 응용 프로그램을 업데이트 합니다.

### <a name="disk-encryption-blocking-upgrade"></a>디스크 암호화에 대 한 차단 업그레이드

응용 프로그램의 암호화 기능 업그레이드를 차단합니다. Windows를 업그레이드 하 고 업그레이드 후 사용 하도록 설정 하기 전에 암호화 기능을 사용 하지 않도록 설정 합니다.

### <a name="does-not-work-with-new-os-but-wont-block-upgrade"></a>새 OS를 사용 하 여 작동 하지 않습니다 하지만 업그레이드를 차단 하지 않습니다

응용 프로그램을 새 OS 버전과 호환 되지 않습니다 하지만 업그레이드를 차단 하지 않습니다. 조치가 계속 하려면 업그레이드에 필요 하지 않습니다. 응용 프로그램의 호환 되는 버전을 새 OS 버전을 설치 합니다.

### <a name="does-not-work-with-new-os-and-will-block-upgrade"></a>업그레이드를 차단 하 고 새 OS를 사용 하 여 작동 하지 않습니다

응용 프로그램이 새로운 OS 버전과 호환 되지 않습니다 및 업그레이드를 차단 합니다. 업그레이드 하기 전에 응용 프로그램을 제거 합니다. 응용 프로그램의 호환 버전을 사용할 수 있습니다.

### <a name="evaluate-application-on-new-os"></a>새 OS에서 응용 프로그램 평가

Windows 응용 프로그램을 마이그레이션하는 있지만 새 OS 버전에서 앱의 성능에 영향을 줄 수 있는 문제 감지 합니다. 조치가 계속 하려면 업그레이드에 필요 하지 않습니다. 새 OS 버전에서 응용 프로그램을 테스트 합니다.

### <a name="may-block-upgrade-test-application"></a>업그레이드를 차단, 응용 프로그램을 테스트할 수 있습니다.

Windows는 업그레이드에 방해가 될 수 있습니다 하지만 추가 조사가 필요한 문제를 감지 했습니다. 업그레이드 하는 동안 응용 프로그램의 동작을 테스트 합니다. 업그레이드를 차단 하는 경우 업그레이드 하기 전에 제거 합니다. 그런 다음 다시 설치 하 고 새 OS 버전에서 테스트.

### <a name="multiple"></a>여러 개

여러 문제는 응용 프로그램을 영향을 줍니다. 선택 **쿼리** Windows에서 발견 된 문제에 대 한 세부 정보를 확인 합니다.

### <a name="reinstall-application-after-upgrading"></a>응용 프로그램 업그레이드 한 후 다시 설치

응용 프로그램 새 OS 버전과 호환 되지만 Windows를 업그레이드 한 후 다시 설치 해야 합니다. 업그레이드 프로세스는 응용 프로그램을 제거 합니다. 조치가 계속 하려면 업그레이드에 필요 하지 않습니다. 새 OS 버전에서 응용 프로그램을 다시 설치 합니다.


## <a name="ready-for-windows"></a>Windows에 대 한 준비

합니다 [준비에 대 한 Windows](https://www.readyforwindows.com) 응용 프로그램 카탈로그에서 상관 관계를 지정 하는 다음 데이터 원본:

- 동일한 앱을 보고 하는 다른 고객 으로부터 진단 데이터
- Microsoft에서 추가 검사 같은 장치에서 호환성 블록

가능한 범주는 다음과 같습니다.

- **많이 채택**: 상용 Windows 10 장치를 100,000 개 이상의이 앱을 설치 합니다.  

- **채택**: 10,000 개 이상의 상업용 Windows 10 장치의이 앱을 설치 합니다.  

- **데이터가 부족 하 게**: 상용 너무 적은 Windows 10 장치를 채택을 분류 하는 Microsoft에 대 한이 앱에 대 한 정보를 공유 합니다.

- **개발자에 게 문의**: 이 버전의 앱과의 호환성 문제가 있을 수 있습니다. 자세한 내용은 소프트웨어 공급자에 게 문의 하는 것이 좋습니다. 자세한 내용은 [준비에 대 한 Windows](https://www.readyforwindows.com/)합니다.  

- **알 수 없음**: 이 응용 프로그램의이 버전에 사용할 수 있는 Windows 준비에 대 한 정보가 없는 합니다. 정보는 응용 프로그램의 다른 버전에 대 한 못할 [준비에 대 한 Windows](https://www.readyforwindows.com/)합니다.  

### <a name="support-statement"></a>지원 문의

소프트웨어 공급자를 지 원하는 경우이 응용 프로그램의 하나 이상의 버전에서 Windows 10 앱 속성 창에서이 문이 표시 됩니다. 호환성 위험 요소 섹션에서 확인 합니다 **문을 지원**합니다.



## <a name="advanced-insights"></a>고급 정보

데스크톱 Analytics도 다음의 추가 정보를 사용 하 여 문제를 검색할 수 있습니다.

### <a name="adopted-version-available"></a>사용 가능한 채택된 버전

다른 고객이 많이 채택 되는이 앱의 다른 버전이 있습니다. 이 신호는 Windows에 대 한 준비에서 데이터를 사용합니다. 현재 버전을 사용 하 여 모든 업그레이드 블 로커 인 경우에 대신 대체 버전을 배포 하는 것이 좋습니다.

### <a name="driver-dependency"></a>드라이버 종속성

앱은 드라이버에 따라 달라 집니다. 데스크톱 분석 파일럿 테스트를 위해 앱을 권장합니다. 파일럿에 대 한 제대로 작동 해야 하지만 회귀를 찾을 수 있습니다. 를 문제가 있는 경우에 Windows 10과 호환 되는 버전을 요청 하려면 게시자에 게 문의 합니다.



## <a name="driver-risk-assessment"></a>드라이버 위험 평가

드라이버에서 데스크톱 Analytics에서 평가 찾을 수 있습니다. 배포 계획의 드라이버 자산 목록에서 해당 속성 플라이 아웃 창을 열려면 개별 드라이버를 선택 합니다. 전체 권장 사항 및 평가 수준을 볼 수 있습니다. 합니다 **호환성 위험 요소** 섹션에서는 이러한 평가 대 한 세부 정보를 보여 줍니다.
