## <a name="bkmk_posh"></a> 향상된 PowerShell 스크립트 실행 작업 순서 단계 기능
<!--1359389--> 이제 **PowerShell 스크립트 실행** 단계에 Windows PowerShell 코드를 직접 입력할 수 있습니다. 이 변경을 통해 먼저 스크립트를 사용하여 패키지를 만들고 배포하지 않고 작업 순서 중 PowerShell 명령을 실행할 수 있습니다.
